# 🛒 E-Commerce Sales Dashboard

Overview
An end-to-end data analytics project analyzing online retail 
sales data using Python and Power BI.

📊 Dashboard Pages
- Sales Overview - KPIs, revenue trends, top countries
- Product Performance - Top products by revenue & quantity
- Customer Segmentation - RFM analysis with 6 segments
- Time Analysis - Revenue patterns by month, day and hour

🛠️ Tools Used
- Python (Pandas) - Data cleaning & RFM segmentation
- Jupyter Notebook - Exploratory data analysis
- Power BI - Interactive dashboard

📁 Project Structure
- `data/raw` - Original dataset (Online Retail)
- `data/cleaned` - Cleaned data & RFM segments
- `notebooks` - Jupyter analysis notebook
- `Ecommerce_Sales_Dashboard.pbix` - Power BI dashboard

📈 Key Insights
- Total Revenue: £5.29M across 17.28K invoices
- 4,228 unique customers across 10+ countries
- UK dominates with 80%+ of total revenue
- Peak sales in November (Christmas season)
- Most orders placed on Thursdays around 12pm
- 488 Champion customers driving majority of revenue
